Class: ModelUpdater

bcdui.core. ModelUpdater

A concrete subclass of TransformationChain, replacing its targetModel's content with the result of the transformation applied to it. Can be applied to all concrete subclasses of AbstractUpdatableModel, like StaticModel or SimpleModel Technically, this is a bcdui.core.TransformationChain object but it should not be executed, fired, modified or read from directly.

args Object An argument object with the following properties:
Name Type Default Description
chain chainDef The definition of the transformation chain
  • a single string with the URL of the transformation XSLT or doTjs template
  • or a JS transformator function
  • or an array with a mixture of URLs and JS transformators
  • or a DataProvider with an XML document following xsd http://www.businesscode.de/schema/bcdui/chain-1.0.0
targetModel bcdui.core.DataProvider The model to be transformed and replaced
parameters Object optional An object, where each property holds a DataProvider, used as a transformation parameters. Once this ModelUpdater is executed, it will check each parameter and execute it, if it is not .isReady()
autoUpdate boolean true optional A boolean value indicating if the ModelUpdater should run on every change in the targetModel. Can be a data modification event or if targetModel again reaches the ready status. If autoUpdate is false a model updater only runs when the targetModel is (re)executed.
id string optional Globally unique id for use in declarative contexts
